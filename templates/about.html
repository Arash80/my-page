About This Blog

Practical lessons from building software.

Arash's Blog explores Python, Flask, web security, testing, and the everyday decisions that turn a learning project into maintainable software.

Each article focuses on useful techniques you can apply immediately. The goal is to explain not only what code to write, but why a design is safer, clearer, or easier to operate.

This website is also an open portfolio project. Its source includes environment-based configuration, automated tests, protected admin actions, sanitized rich text, and a reproducible content-seeding command.
